# Plantillas de actas

Coloca aquí tu plantilla PDF de acta de entrega/recepción con el nombre:

```
plantilla-acta.pdf
```

El backend la leerá desde esta carpeta en cada solicitud de impresión
(no requiere recompilar ni reiniciar la app para actualizar la plantilla).

Pendiente: una vez que compartas el PDF, se define aquí la lista de
campos de formulario (AcroForm) que el backend rellena automáticamente
por cada acta (empleado, activo, usuario TI, fechas, estado, motivo).
